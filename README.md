# Personal_RPG
This is a passion project meant to be my first real attempt at making a video game and keep my java programming skills largely fresh. No AI or textbook is being used to make this code. All of the code is originally done by me, Alexander Winfield. Currently the plan is to make a simple little japanese-style roleplaying game where the player must manage resources and search a nearby cave for some sort of treasure. Earlier this year, I tried to make a GUI (using JSwing) at the same time I was planning out all the background calculations and the work started to overwhelm me, so I'm planning to make the game mainly console based and getting the classes and calculations figured out and the game functional, before I start worrying about anything GUI related.

The game only needs a simple java development kit to run with Main.java having the main method/function. To explain how it works from a high-level standpoint, I use Stats.java to track the player's and enemy's stats, Inventory.java to keep track of the potions the player is holding (as well as any additional items I might want to add in the future), Combat_Script.java to handle battles that happen as the player searches the cave, and Main_Script.java to handle the player interacting with main areas, such as the cave or the main hub area "Bell Town".
